🚀 ¿Qué puede hacer KósmOS?

Estas son las funcionalidades principales del sistema operativo galáctico KósmOS:

  • 🛰️ Shell personalizada: una terminal cósmica con comandos únicos como orbita, estrella y agujero, además de comandos clásicos como cd, mkdir y exit.
  • 🖥️ Interfaz Gráfica Galáctica: escritorio minimalista con ventanas flotantes, menús interactivos y fondos inspirados en el universo.
  • 📚 Manual integrado: documentación completa del sistema accesible vía terminal usando lynx, ideal para exploradores curiosos.
  • 📂 Exploración de archivos: navega por tu universo de archivos y programas simulados como si fueras un astronauta digital.
  • 🔗 Conexión entre módulos: la shell, la UI y el manual se integran para ofrecer una experiencia fluida, moderna y funcional.
